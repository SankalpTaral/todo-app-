# Todo App (Vanilla JavaScript)

A simple Todo application built using **Vanilla JavaScript** that allows users to manage daily tasks efficiently.

## Features
- Add new todos
- Edit todos inline
- Mark todos as Finished / In Progress
- Delete todos
- Filter pending todos
- Dynamic numbering
- Clean UI with DOM manipulation

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## Learning Outcomes
- DOM manipulation
- Event handling
- State management using UI
- JavaScript arrays and loops
- Clean separation of CSS and JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in a browser
